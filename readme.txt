Compilado no Windows 10 x64 com MingW.
Nome dos arquivos para indexação estão contidos em "arquivos.txt" e podem ser alterados livremente (um arquivo por linha).
Não suporta palavras com acento e caracteres especiais (remove os mesmos das palavras indexadas).
Arquivo "word generator.cpp" é usado na criação de arquivos com grande número de palavras. Utilizado nesse projeto para teste de software.
